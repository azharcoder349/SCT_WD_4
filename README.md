# 📋 Advanced To-Do Web Application

An advanced and interactive To-Do Web Application developed using **HTML, CSS, and JavaScript**. This project helps users efficiently manage daily tasks with a modern Glassmorphism UI, responsive design, and interactive features.

---

## 🚀 Features

- ➕ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Completed
- 📅 Set Due Date
- ⏰ Set Due Time
- 🔴🟡🟢 Priority Levels (High, Medium, Low)
- 🔍 Search Tasks
- 📂 Filter Tasks (All, Pending, Completed)
- 📊 Progress Tracker
- 📈 Dashboard Statistics
- 🌙 Dark Mode
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- ✨ Modern Glassmorphism User Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
